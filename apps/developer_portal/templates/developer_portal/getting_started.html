Getting started

Make your first API request in under 5 minutes.

{# Step 1 #}
1

Get an API key

Sign in to your account and navigate to Settings → API Keys. Click Create API key, give it a name, and copy the key — it won't be shown again.

{% if user.is_authenticated %} Create an API key → {% else %} Sign in to create an API key {% endif %}
{# Step 2 #}
2

Make your first request

Pass the key in the Authorization header:

bash 
curl https://{{ request.get_host }}/api/v1/accounts/me/ \
  -H "Authorization: Api-Key YOUR_API_KEY"
